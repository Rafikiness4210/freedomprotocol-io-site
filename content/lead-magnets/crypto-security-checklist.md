# The Crypto Security Checklist
## 25-Point Audit You Can Run in 30 Minutes

*By FreedomProtocol.io — Updated March 2026*

---

### Wallet Security (7 points)
- [ ] Hardware wallet firmware is up to date
- [ ] Seed phrase stored on metal (not paper, not digital)
- [ ] Seed phrase in 2+ separate physical locations
- [ ] Hot wallet holds < 5% of total holdings
- [ ] No unused token approvals (check revoke.cash)
- [ ] Wallet apps downloaded from official sources only
- [ ] Using a dedicated device for crypto transactions

### Account Security (5 points)
- [ ] Unique email for crypto accounts (not your personal email)
- [ ] 2FA enabled on ALL exchange accounts (hardware key preferred)
- [ ] No SMS-based 2FA (SIM swap vulnerable)
- [ ] Unique strong password per exchange (password manager)
- [ ] Withdrawal address whitelisting enabled

### Device Security (5 points)
- [ ] Device OS and apps fully updated
- [ ] Full disk encryption enabled
- [ ] Antivirus/anti-malware active and current
- [ ] No browser extensions except verified security tools
- [ ] VPN active when accessing crypto services

### Network Security (4 points)
- [ ] Never access wallets on public WiFi (even with VPN)
- [ ] Home router firmware updated
- [ ] DNS set to encrypted provider (1.1.1.1 or Quad9)
- [ ] Using a no-log VPN for all crypto activity

### Operational Security (4 points)
- [ ] Never publicly discussed exact holdings
- [ ] No screenshots of wallet balances shared
- [ ] Exchange withdrawal confirmations go to secure email
- [ ] Emergency plan documented (what happens if you're incapacitated)

---

### Scoring
- **25/25:** Protocol-grade security. Maintain it.
- **20–24:** Strong. Fix the gaps this week.
- **15–19:** Vulnerable. Prioritize hardware wallet + 2FA immediately.
- **Below 15:** Critical risk. Stop everything and secure your stack today.
